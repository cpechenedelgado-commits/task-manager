# Task Manager

Aplicación web para gestionar tareas diarias.

## Funcionalidades
- Crear, editar y eliminar tareas
- Marcar tareas como completadas
- Filtrar tareas
- Persistencia con LocalStorage

## Tecnologías
- HTML
- CSS
- JavaScript (ES6+)

## Aprendizajes
- Manejo del DOM
- Estado de la aplicación
- Arquitectura básica en JavaScript
